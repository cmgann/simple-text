Text Replace

Data Field:

Type your data in the Data field for the text that you would like to manipulate.

Example:
John,Doe
Jane,Doe

Be sure to verify that your Separators are accurate. For this we will use:
Column Separator = ,
Row Separator = /n

** Hint, the /n means that it is for the next line, we will go over this more below in the Separator Help section. **

Pattern Field:

Type your pattern for which you would like to the "$" to tag where you would like to insert the text from your Data Field.

Example:
First Name: $0
Last Name: $1

This will have the Result:
First Name: John
Last Name: Doe

First Name: Jane
Last Name: Doe

*** Important, the first tag will start with $0 the next tag would be $1 and so on. ***

Separator Help:

Some Separators will need to start with a '\' some examples below:
\n = New Line
\t = Tab
\| = | (Pipe)

Most normal characters will not require '\'.

You can save your pattern for future use!

If you have a pattern added in that you would like to save for your common tasks, then click on the Menu button to the right of the dropdown and click "save".

Then later on when you visit you can select it from the drop down menu and it will automaticly populate your pattern for you.
